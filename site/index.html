Shared Architecture

One architecture.
Every instrument.

Every μ-series instrument shares the same mixer and modulation architecture across all 8 tracks. Whatever the sound engine, the same powerful modulation and mixing system is underneath.

Eight MSEG or stepped-sequence modulators per track route to any parameter. All tracks flow into a full mixer with per-channel sidechain — any track ducks any other, or from an external host signal — three send effects, and two master insert slots.

Individual outputs are available per track, and any sequence or voice can be swapped live — manually or via MIDI Program Change — without stopping playback.

Live Sequencer Swap

Switch any sequence or voice while playing via MIDI Program Change. In μ-clid these are Rhythms; in μ-tant, Voices.

Sidechain on Every Channel

Every mixer channel has its own sidechain input — duck any channel from any other, or from an external host signal. Use a μ-clid kick to pump a μ-tant drone pad for classic electronic textures.

Individual Outputs

Route each track to its own DAW output for parallel processing, hardware effects, or external mixing.
