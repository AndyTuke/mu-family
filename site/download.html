macOS & Linux — Beta

Beta version — please feed back any issues you have using these. Universal macOS (Apple Silicon + Intel) and Linux builds, version v1.0.0.926. They're new, so anything you hit helps — email support@transwarp.me.

μ-clid macOS ↓ Linux ↓ μ-clid LITE (free) macOS ↓ Linux ↓ μ-tant macOS ↓ Linux ↓

macOS isn't notarized yet — after unzipping, run xattr -dr com.apple.quarantine on the bundle (or right-click → Open) to get past Gatekeeper.

All downloads are .zip archives containing the VST3, CLAP (and Standalone where applicable). Unzip and copy the plugin into your system plugin folder. Already bought a licence? Open the plugin, click the demo banner, and paste your key to activate. Trouble installing? Contact support.
