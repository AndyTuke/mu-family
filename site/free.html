μ-clid LITE
Single-Track Euclidean Sequencer · VST3 · CLAP · Standalone · Windows

μ-clid LITE gives you one complete rhythm track — the full euclidean algorithm, MIDI output, and the complete workflow — permanently free. No expiry date, no save restrictions.

It's the ideal way to learn the euclidean approach before committing to the full 8-track version. Everything you learn in LITE transfers directly to the full version.

  • Single rhythm track — fully featured
  • Full euclidean algorithm (steps, hits, rotate)
  • MIDI note output — works with any DAW instrument
  • VST3 · CLAP · Standalone · Windows
  • No time limit · Permanent install · No registration required

μ-link
Sync & Performance Hub · Standalone · Windows · Free

μ-link connects multiple μ-series instances together into a unified performance tool. Run μ-clid and μ-tant simultaneously — μ-link keeps their BPM and transport locked together in real time.

A lightweight standalone app designed for live performance. No DAW required. Launch your μ-series instruments, connect them through μ-link, and perform as a single coherent system.

  • BPM sync across all connected μ-series instances
  • Transport sync — start, stop, and position shared
  • Lightweight standalone app — minimal CPU footprint
  • No DAW required — built for live performance
  • Windows · Free forever
