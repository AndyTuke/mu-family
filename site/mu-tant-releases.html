μ-tant — what's new

Every change that affects you, version by version. Numbers in brackets are our internal tracking references.

New

  • Master Loop control in the transport bar (like μ-clid) — pick a loop length and see the step counter. (1010)
  • Hot-swap timing setting — choose whether preset / program-change swaps land on the master loop or on each voice's own loop, for musically-timed changes. (1011)
  • External MIDI clock sync (standalone) — slave μ-tant's tempo and transport to incoming MIDI clock. Set it under Settings → MIDI Clock. (1012)
  • Note mode — play μ-tant like an instrument: the drone sounds only while a MIDI note is held, and the held note sets the pitch (works in a DAW and from a keyboard in standalone). Switch it on under Settings → Note Mode. (1012)
  • Redesigned X-Mod — two lanes instead of three fixed knobs. Lane A: one Index knob with FM / PM / TZFM modes, plus Sync and Feedback. Lane B: a bipolar Depth knob with AM, RM and SSB (single-sideband frequency-shift) modes — from tremolo-like amplitude mod through clangorous ring mod to shimmering inharmonic drones. Existing presets are carried over automatically. (1019)
  • The plugin now tells you when a newer version is available — a banner appears along the bottom showing your version and the latest one, click it to go to the download page. (1026)
  • 100 new factory voice presets (drones, pads, basses, leads, textures, rhythmic) and 100 full presets combining them with effects — in the Voices and Presets browsers. (1029)
  • All filter controls — cutoff, resonance, drive and low-cut — for both filters are now modulation destinations, grouped as Filter 1 and Filter 2 in the destination list. (1030)
  • Pitch-envelope depth (per oscillator) and filter-envelope depth (both filters) are now modulation destinations too. (1034)

Fixed

  • Fixed a potential click/dropout when using modulation — the modulation engine no longer allocates memory on the audio thread. (1039)
  • Fixed a potential click/dropout when automating reverb or effect parameters — these no longer allocate memory on the audio thread. (1040)
  • The in-plugin update check now runs once per session instead of on every window open. (1038)
  • Aligned the Close button on the MIDI program-change preset tables. (1032)
  • Fixed garbled characters in the MIDI program-change "Full Presets" table heading. (1014)
  • Fixed the filter envelope (gate editor's FILT layer) barely moving the cutoff — it now follows the drawn shape and sweeps the filter as intended. (1020)
  • Pitch now glides smoothly — the pitch envelope and smooth modulators sweep continuously instead of stepping in semitones. Step modulators still snap to semitones so you can write melodies. (1021)
  • Step modulators now snap their drawing grid to the target's musical units — an octave target gives 3 steps up/down, a scale-degree target 12 — so you draw notes directly instead of guessing heights. (1022)
  • Octave is now a step-only modulation target — it's hidden when a modulator is in Smooth mode, since a continuous octave glide isn't musical. (1023)

Improved

  • The factory presets have been completely reworked — every voice now has its own evolving gate pattern (no more flat held drones), the rhythms are more musical and leave space so layered voices play off each other, and the combined presets pair voices that complement one another. (1042)
  • Switching to a preset with fewer voices while playing now fades the removed voices out smoothly, so their filter/effect tails ring out instead of cutting off. (1013)
  • The Settings page has been restyled to match μ-clid — grouped General and MIDI sections with clear headings, dividers and a consistent layout. (1004)
  • Settings controls are now left-aligned and indented under their headings, with a Contact Support link at the bottom. (1031)
  • Oscillator knobs reordered — the Pitch Env knob now sits with the pitch controls (Oct / Semi / Fine) and wavetable Position is last, so it no longer reads as a "position envelope". (1033)
  • The modulator destination list now hides the Insert effect targets when no insert effect is selected. (1035)
v1.0.0.865 released 13 June 2026 Current

Improved

  • In a DAW, the transport play button now turns into a stop icon while the host is playing — matching the standalone. (994)
  • Transport bar polish: clearer play/stop state and tidier spacing and labels. (990, 991)

Fixed

  • Fixed a rare crash on launch that could occur when a saved online-activation file was present. (995)
  • Fixed knobs accidentally changing value when you scroll over them in a DAW (e.g. while scrolling the timeline). (988)
v1.0.847 released 12 June 2026

New

  • Demo mode: try the full synth with a single voice; saving is disabled until you activate a licence. (981, 982)
  • The sidechain input now appears in CLAP hosts such as Bitwig, matching VST3. (969)

Improved

  • New modulators now start in Unipolar mode. (968)
  • The modulator step grid now lines up with the actual step length. (963, 967)
  • Double-click a modulator node to delete it. (962)
  • Modulation curves no longer overshoot the visible range when bending. (957)
  • The master meter and fader now reflect the master insert effects. (965)

Fixed

  • External (DAW) sidechain now ducks correctly — previously it could produce no ducking at all. (974)
  • Fixed a possible crash at high sample rates (88.2 kHz and above) in the Chorus effect. (961)
